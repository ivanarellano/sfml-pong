# Dependencies
- Setup SFML 2.4.2 through their [instructions](https://www.sfml-dev.org/tutorials/2.4/#getting-started).
- Download [sfml-pong 1.0](https://github.com/ivanarellano/sfml-pong/releases/download/1.0/sfml_pong_1_0.zip), open it, and copy `OpenAL32.dll` to the directory where your compiled executable is. 
- [FFF Forward Font](http://www.1001fonts.com/fff-forward-font.html). This font should be placed in the `assets` folder with the filename `FFFFORWA.TTF`.
